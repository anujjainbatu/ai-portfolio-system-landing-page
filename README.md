# 🚀 Portfolio Builder Landing Page

A highly optimized, SEO-friendly landing page for the revolutionary AI-powered portfolio builder. This landing page is designed to rank for high-traffic keywords and convert visitors into users.

![Portfolio Builder Landing Page](assets/landing-preview.png)

## 🎯 Purpose

This landing page serves as the marketing front-end for the [Portfolio Builder project](https://github.com/anujjainbatu/portfolio), targeting developers, job seekers, and professionals who want to create stunning portfolios without coding.

## ✨ Features

- **🔍 SEO Optimized**: Complete meta tags, structured data, and keyword optimization
- **📱 Mobile First**: Responsive design that looks great on all devices
- **⚡ Fast Loading**: Optimized performance with lazy loading and compressed assets
- **🎨 Modern Design**: Professional, conversion-focused design
- **🤖 AI Integration**: Showcases AI chatbot capabilities
- **📊 Analytics Ready**: Built-in tracking and conversion optimization

## 🎨 Customization

### Colors and Branding
Update the CSS custom properties in the `<style>` section:

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --accent-color: #10b981;
  --text-color: #333333;
  --background-color: #ffffff;
}
```

### Content Updates
1. **Headlines**: Update in the hero section
2. **Features**: Modify the features grid
3. **Testimonials**: Replace with real user feedback
4. **Links**: Update all GitHub and demo links
5. **Contact Info**: Update social media and contact links

### Images
Replace images in the `assets/` folder:
- `demo.mp4` - Product demo video
- `landing-preview.png` - Social media preview
- `favicon.ico` - Browser icon

## 📈 Analytics & Tracking

### Google Analytics 4
Add your GA4 tracking code:

```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_TRACKING_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_TRACKING_ID');
</script>
```
