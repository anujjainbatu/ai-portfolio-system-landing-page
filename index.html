How It Works

Three simple steps to your perfect portfolio

1. Configure

Edit the portfolio-config.json file with your information. Use our AI assistant or fill it manually - it's just like filling out a form.

git clone repo
Edit portfolio-config.json
Add your images to /public

2. Customize

Add your API key for the AI chatbot, upload your profile picture and project screenshots. Everything is automatic from there.

npm install
Add API key to .env
npm run dev

3. Deploy

Deploy with one click to Vercel, Netlify, or any hosting platform. Your portfolio goes live instantly with your custom domain.

npm run build
Deploy to Vercel
You're live!

Frequently Asked Questions

Do I need coding skills to use this?

Not at all! You just need to edit a JSON file (like filling a form) and add your images. No HTML, CSS, or JavaScript knowledge required.

Is it really free?

Yes, completely free and open source. You only pay for hosting (which can be free on Vercel/Netlify) and optionally for the AI API if you want the chatbot feature.

How do I add the AI chatbot?

Simply get a free Google Gemini API key, add it to your environment variables, and the chatbot will automatically work with your portfolio data.

Can I customize the design?

Absolutely! The portfolio is built with Tailwind CSS and React components. You can modify colors, layouts, and add new sections as needed.

What if I need help setting it up?

We have detailed documentation, video tutorials, and an active community. You can also open issues on GitHub for support.
