How It Works

Three simple steps to your perfect portfolio

1. Configure

Edit the portfolio-config.json file with your information. Use our AI assistant or fill it manually - it's just like filling out a form.

git clone repo
Edit portfolio-config.json
Add your images to /public

2. Customize

Add your API key for the AI chatbot, upload your profile picture and project screenshots. Everything is automatic from there.

npm install
Add API key to .env
npm run dev

3. Deploy

Deploy with one click to Vercel, Netlify, or any hosting platform. Your portfolio goes live instantly with your custom domain.

npm run build
Deploy to Vercel
You're live!

Frequently Asked Questions

Everything you need to know about our portfolio builder

Do I need coding skills to use this portfolio builder?

Not at all! You just need to edit a JSON file (like filling a form) and add your images. No HTML, CSS, or JavaScript knowledge required. Our portfolio builder is designed for developers and non-developers alike.

Is this portfolio builder really free?

Yes, completely free and open source! You only pay for hosting (which can be free on Vercel/Netlify) and optionally for the AI API if you want the chatbot feature. No hidden costs or subscription fees.

How do I add the AI chatbot to my portfolio?

Simply get a free Google Gemini API key, add it to your environment variables, and the chatbot will automatically work with your portfolio data. The AI chatbot helps visitors learn about your experience and projects interactively.

Can I customize the portfolio design and layout?

Absolutely! The portfolio is built with Tailwind CSS and React components. You can modify colors, layouts, add new sections, and fully customize the design to match your personal brand and style preferences.

What if I need help setting up my portfolio?

We provide detailed documentation, step-by-step video tutorials, and have an active community on GitHub. You can also open issues on our GitHub repository for personalized support from our team and community.

How long does it take to create a portfolio?

Most users have their portfolio up and running in under 5 minutes! This includes editing the JSON configuration file, adding images, and deploying to a hosting platform. It's the fastest way to create a professional portfolio in 2025.
